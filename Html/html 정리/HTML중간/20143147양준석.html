HTML 문서 구조 태그

HTML 문서의 구조를 정의하기 위한 기본 태그들은 다음과 같으며, 필수 태그로 표시된 태그들은 문서에서반드시사용되어야 한다.

<html> : 문서의 시작과 끝(필수 태그)

<head> : 문서 머리 부분의 시작과 끝(필수 태그)

<title> : 문서 제목의 시작과 끝(필수 태그)

<body> : 문서 몸통 부분의 시작과 끝(필수 태그)
